# Welcome to sn

sn is a TUI notes app inspired by Obsidian.

## How to use

- **a**: create a new note (opens a name prompt)
- **i**: edit the selected note in the preview panel
- **Esc**: save changes and leave edit mode
- **Arrow keys** or **j/k**: navigate between notes
- **[** / **]**: scroll note content
- **g**: jump to the first note
- **G**: jump to the last note
- **q**: quit

## Organization

Place `.md` or `.txt` files in the `notes/` directory.
Each file appears in the left panel list.
